CBPR - Help
What is CBPR?

Colour blind page renderer or CBPR is a chrome extension that assists users in viewing web pages that may be difficult to read. In essence, this extension is repainting the entire page to a colour scheme that suits you.

Colour changer

Colour changer will take any pre-set configs or default config and depending on which has been selected will show a preview of the selection. From there, clicking the button on the left side of the page will transform the web-page you are currently viewing.

Custom designs

This section of the extension is for creating any combination of colours that you may think will help you read the web-page. When one of the pre-set configs is chosen you may modify the name, background colour, font colour and border colour. There is a small preview window in which you can see what it will look like on the page. This is achieved using chrome storage.

Help and other
