Help.
This extension will change the background colors that may aid some that have disabled eyes.

Magic tool.
The magic tool can help you catch the spots that we missed, simply click on the arrow then click on the part of the page you want to change. use your left mouse button to use the primary color or use the secondary mouse button to use the secondary color.

Color Picker.
The Color Picker tool allows you to choose the variant of colorblindness you are affected by and the site that you are on will change the colors that are best suited for that variant.

Magic tool.
Click the arrow to start the magic tool.
⇦
Then click any element on the screen and it will change it to your default primary color.
